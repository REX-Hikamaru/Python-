基本的なPythonの概念

Pythonは多目的プログラミング言語です。いくつかの基本的な概念を紹介します：


# こんにちは、世界
print("Hello, World!")

# 変数
x = 5
y = "こんにちは"

中級者向けPythonの概念

進むにつれて、より複雑な構造に出会うことになります：


# 関数
def greet(name):
    return f"こんにちは、{name}さん！"

print(greet("アリス"))

# リスト
fruits = ["りんご", "バナナ", "チェリー"]
for fruit in fruits:
    print(fruit)

上級者向けPythonの概念

上級トピックには、オブジェクト指向プログラミングやデコレーターが含まれます：


# クラス
class Dog:
    def __init__(self, name):
        self.name = name

    def bark(self):
        return "ワン！"

my_dog = Dog("バディ")
print(my_dog.bark())

# デコレーター
def decorator_function(original_function):
    def wrapper_function():
        print("ラッパーが{}の前に実行されました".format(original_function.__name__))
        return original_function()
    return wrapper_function

@decorator_function
def display():
    return "表示関数が実行されました。"

print(display())

Pythonとターミナルの基本コマンド

Pythonの起動・終了や、基礎から応用までのターミナルコマンド例です。


# Pythonインタプリタの起動
python

# Pythonインタプリタの終了
exit()
# または
quit()

# Pythonファイルの実行
python ファイル名.py

# 仮想環境の作成
python -m venv venv

# 仮想環境の有効化（Windows）
venv\Scripts\activate

# 仮想環境の無効化
deactivate

# 必要なパッケージのインストール
pip install パッケージ名

# インストール済みパッケージの一覧表示
pip list

# requirements.txtから一括インストール
pip install -r requirements.txt

# Flaskアプリの起動（例: run.pyがある場合）
python run.py

# ファイル一覧表示（Windows）
dir

# ディレクトリ移動
cd ディレクトリ名
